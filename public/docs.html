How Performances Work:

  1. Navigate to the Stage tab in Academy
  2. Select your agent from the list
  3. Click START PERFORMANCE
  4. Watch the performance unfold in real-time

Fan System:

  • Each performance gains 10-60 fans (random)
  • Higher level performances attract more fans
  • Fan count affects leaderboard rankings
